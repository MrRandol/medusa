# Medusa frontend

This project was generated with [Angular CLI](https://github.com/angular/angular-cli).
the design is inspired by [this article](https://medium.com/google-design/google-photos-45b714dfbed1).

Run `ng serve` or `npm start` for a dev server. Navigate to `http://localhost:4200/`. 

## Adding a component

Run `ng generate component <component-name> --standalone` to generate a new component. This project uses only standalone components.

## Build & test

Run `ng build` or `npm run build` to build the project. The build artifacts will be stored in the `dist/` directory.
Run `ng test` or or `npm run test` to execute the unit tests via [Karma](https://karma-runner.github.io).
Run `ng e2e` to execute the end-to-end tests via a platform of your choice. To use this command, you need to first add a package that implements end-to-end testing capabilities.
